# QPastebin

> A script to quickly paste text or files to [pastebin.com](http://pastebin.com)

1. Run:

  * Upload text
  ```sh
  qpaste "Hello world!"
  ```
  * Upload a file
  ```sh
  qpaste mycode.cpp
  ```

  > (You can also directly copy the outputted link via `xcopy`)

2. Install:

  * Add to path:
  ```sh
  sudo mv qpaste.sh /bin/qpaste
  chmod +x /bin/qpaste
  ```
